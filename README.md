
# Asperboy: Jogo de Aprendizado para crianças autistas

É um jogo na plataforma Roblox que tem como objetivo conscientizar sobre a sindrome de asperger, e tambem ajudar pessoas com sindrome de asperger a como proceder em certas situações


## Softwares utilizada

**Front-end:** Roblox studio, Blender, Reaper, krita

**Back-end:** Roblox studio(lua)

**Controle-de-Versão:** Roblox studio

**Testes-e-Validação:** Roblox studio
